# 思考与成长

这里记录学习过程中的方法总结、阶段复盘与一些较长期的思考。

## 为什么要单独保留这个部分

课程和项目更多是在记录“我学了什么、做了什么”。  
而思考部分更关注：

- 我是怎么理解这些内容的
- 我在哪些地方走了弯路
- 我有哪些方法逐渐有效
- 我对学习、项目和科研训练的认识发生了什么变化

## 这一部分会写什么

### 学习方法
包括复习方法、整理方法、知识结构建立方法等。

### 阶段复盘
包括某门课、某个项目或某一阶段后的总结与反思。

### 路径理解
包括对保研准备、科研训练和长期成长路线的一些认识。

## 我希望它起到什么作用

如果说“学习”记录知识，“项目”记录实践，  
那么“思考”就是把这些经历重新组织起来，使它们逐渐形成更稳定的方向感与成长意识。
