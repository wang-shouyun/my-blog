# 诗歌

这里保存我的诗词创作与相关鉴赏解释。

对我来说，诗歌并不是与学习、项目、思考完全割裂的另一部分。  
它更像是另一种表达方式：当知识试图建立结构，项目试图解决问题，思考试图整理经验时，诗歌则试图安放那些更难直接说清的感受、哲理与时间意识。

## 这里会包含什么

### 我的诗词
收录我自己创作的诗歌、词作与阶段性作品。

### 鉴赏与解释
对部分作品进行背景说明、意象解释、主题分析与结构梳理，也记录我对诗歌表达的理解。

## 我希望这一部分成为什么

我希望这里不仅是作品陈列区，也是一块能留下语言探索、审美追求与精神轨迹的地方。  
如果说“学习”记录知识，“项目”记录实践，“思考”记录反省，那么“诗歌”记录的，就是那些更接近心灵、历史感与生命感的部分。
