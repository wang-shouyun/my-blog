# 项目与实践

这里记录我的项目实践、实验尝试和阶段性成果整理。

## 我为什么重视项目

课程告诉我“知识是什么”，  
项目则让我理解“知识如何真正被使用”。

因此，我希望把项目不只记录成做过的事情，而是整理成能复盘、能表达、能展示能力的过程材料。

## 当前整理方向

### 1. 调制识别相关项目
围绕自动调制分类、数据处理、模型训练与实验分析展开。

### 2. 智能侦测 / 信号处理相关尝试
记录信号分析、无线相关实践或系统设计中的参与内容与技术路线。

### 3. 课程 / 实验 / 竞赛实践
把课程设计、实验项目和比赛经历逐步整理成完整记录。

## 每个项目我希望写清楚什么

- 背景
- 目标
- 我的任务
- 方法
- 实验与过程
- 难点与问题
- 最终收获
