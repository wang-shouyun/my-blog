# 关于我

你好，我是王守云。

这是我的个人主页与学习博客，用来记录课程笔记、项目经历、阶段反思以及长期成长过程中的一些思考。

## 我关注的方向

- 通信基础理论与相关课程学习
- 模式识别与智能信息处理
- 信息论与信号分析方法
- 项目实践、科研训练与长期能力积累

## 我为什么搭建这个博客

对我来说，这里不只是一个存放资料的地方。  
我更希望它成为一个持续积累的空间：把课程中的知识、项目中的经验、学习中的思考慢慢组织起来，最后沉淀为真正属于自己的理解、表达和能力。

## 这个博客会包含什么

### 学习
记录课程框架、知识点整理、公式推导、典型题目和复习总结。

### 项目
记录做过的项目、实验和技术尝试，包括背景、方法、过程、问题和收获。

### 思考
记录学习方法、课程复盘、阶段反思，以及对科研训练和成长路径的一些理解。

## 我希望它最终成为什么

我希望这里能逐渐形成一条完整的个人轨迹：

从课程学习出发，  
通过项目实践深化理解，  
再经由反思与表达，把知识逐步转化为更稳定的能力与更清晰的方向。
