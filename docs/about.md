# 关于我

你好，我是王守云。

这是我的个人主页与学习博客，用来记录课程笔记、项目经历、科研探索以及阶段性的思考总结。

## 我的关注方向

- 通信基础理论与相关课程学习
- 模式识别与智能信息处理
- 信息论与信号分析方法
- 项目实践、实验设计与长期能力积累

## 这个博客会写什么

### 1. 课程笔记
我会持续整理通信原理、模式识别、信息论等课程内容，把知识点从“看过”逐渐变成“理解并能表达”。

### 2. 项目记录
我会把做过的项目、实验和技术尝试整理成较完整的记录，包括背景、方法、过程、问题和收获。

### 3. 学习总结
这里也会保存阶段复盘、复习框架、方法总结，以及一些关于学习路径和能力成长的思考。

## 我希望它变成什么

我希望这里不是单纯堆放资料的地方，而是一条能长期积累的个人轨迹：  
从课程学习出发，逐步连接项目实践、科研训练与表达能力，最后沉淀成真正属于自己的知识结构与成长证据。
