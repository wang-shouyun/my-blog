# 学习总览

这里是我的学习与课程整理入口。

目前我主要围绕以下几门课程持续积累：

## 1. 通信原理
关注通信系统的基本模型、模拟与数字通信方法、重点公式和典型题型。

## 2. 模式识别
关注课程整体框架、常见分类方法、核心概念、公式推导与复习重点。

## 3. 信息论
关注信息量、熵、互信息、信源编码、信道容量等内容的理解与整理。
